## 三条核心原则

1. **实测高于文档**——游戏文档与真实 API 行为存在大量出入，一切结论以实测为准并标注样本量
2. **保守优于激进**——在累进惩罚环境中，触限的期望损失远大于赶进度的收益
3. **记忆是工程问题**——Agent 的可靠性不取决于聪明，而取决于状态能否无损穿越会话边界
